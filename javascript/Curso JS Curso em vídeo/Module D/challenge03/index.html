Como é calculada a fórmula de Bhaskara?

  1. Identifique os coeficientes: a, b, c na equação quadrática
  2. Calcule o discriminante (delta)
  3. Verifique o valore de delta:
    • Se delta > 0, a equação tem duas raízes reais distintas.
    • Se delta = 0, a equação tem duas raízes reais iguais.
    • Se delta < 0, a equação não tem raízes reais
  4. Calcule as raízes
  5. Substitua os valores de a, b, c e delta na fórmula de Bhaskara para encontrar as raízes
